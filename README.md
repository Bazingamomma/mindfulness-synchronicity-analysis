# Mindfulness & Synchronicity Analysis

This repository contains a personal data analysis project exploring the relationship between daily mindfulness practices and the number of synchronicities noticed each day.

## Goals
- Track daily mindfulness activities
- Record synchronicities and related factors (mood, stress, sleep)
- Clean and analyze the dataset
- Identify any patterns or correlations
- Practice the full Data Analysis Lifecycle

## Data Elements
- Date
- Minutes of mindfulness practice
- Witness + Spark journaling (yes/no)
- Number of synchronicities noticed
- Mood rating (1–10)
- Stress level (1–10)
- Sleep quality (1–5)
